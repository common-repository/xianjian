== Description ==
先荐是一款快速给网站添加“猜你喜欢”功能的插件。基于深度学习技术的先荐插件，可以提供相关推荐内容和符合用户喜好的内容，最终提升整个网站的粘性和留存。

先荐插件依赖将网站内容和用户行为数据上传到云端（https://*.4paradigm.com/），通过第四范式推荐引擎在云端做计算，并最终将计算推荐结果返回给网站，展示给用户。先荐在以下情况下会请求服务端上报数据或请求服务：
1. 新建和保存推荐栏
2. 上报网站内容到服务器，用于计算并返回推荐结果
3. 上报网站用户行为到服务器，用于计算并返回推荐结果
4. 前端向服务器请求，获取和渲染推荐结果
5. 插件向服务器请求，统计推荐效果数据

先荐插件支持相关推荐、个性化推荐、热门推荐等推荐类型。不同于友荐、无觅、百度推荐，先荐支持设置多个“猜你喜欢”推荐位；信息流配置，更能够将你的网站一键具备今日头条能力。先荐尊重版权，不提倡采集站使用我们的插件。

先荐插件面向用户群体为中国境内网站站长，会严格履行保护网站用户隐私和网站内容安全，（详细用户协议见：https://nbrecsys.4paradigm.com/#/agreement）


 
== Installation ==

1.上传`xianjian` 文件夹到`/wp-content/plugins/` 目录下
2.在wordpress插件菜单里启用先荐插件

 
== Frequently Asked Questions ==
 
= 先荐插件可以设置多个推荐栏吗? =
 
支持设置多个推荐栏
 
= 如何对推荐栏进行样式详细设置? =

点击新建推荐栏页面“设置更多样式”
 



= 推荐栏设置完成之后，多久可以上线?=
 
一般一小时之内就可以上线
